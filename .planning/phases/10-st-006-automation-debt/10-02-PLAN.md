    ```python
    class AutomationLayer(BaseModel):
        model_config = ConfigDict(str_strip_whitespace=True)
        layer_name: str = Field(..., min_length=1, max_length=200)
        layer_type: Literal["quick_fix", "fundamental", "transition"] = Field(...)

                description=f"Automation layer ({self.layer_type}): {self.layer_name}",
                times_applied=self.times_applied,
                is_symptomatic=self.layer_type == "quick_fix",
                side_effects=self.side_effects,
                duration_active=self.duration_active,
            )
    ```

    **DetectAutomationDebtInput model:**
    ```python
    class DetectAutomationDebtInput(BaseModel):
        model_config = ConfigDict(str_strip_whitespace=True, extra="forbid")
        system_description: str = Field(..., min_length=5, max_length=500)
        automation_layers: List[AutomationLayer] = Field(..., min_length=1, max_length=20)
        fundamental_solution: Optional[str] = Field(default=None)
        fundamental_viability_now: Optional[int] = Field(default=None, ge=1, le=10)
        fundamental_viability_original: Optional[int] = Field(default=None, ge=1, le=10)
        response_format: ResponseFormat = Field(default=ResponseFormat.MARKDOWN)
    ```

    **detect_automation_debt tool function:**
    1. Convert AutomationLayers to FixRecords via to_fix_record()
    2. Build DetectBurdenShiftInput with system_description as recurring_symptom, fix_records, and viability fields
    3. Call detect_burden_shift internally with response_format=JSON to get structured data
    4. Parse burden_result JSON
    5. Compute automation-specific additions:
       - `_compute_debt_score(burden_result, layers)`: debt_score = round(burden_result["erosion"]["risk_score"] * 0.6 + automation_factor * 0.4) where automation_factor = min(100, (quick_fix_count / total_count) * 80 + (max_times_applied / 10) * 20)
       - `_identify_erosion_channels(layers, burden_result)`: return list of {name, active, description, ai_example} for Knowledge Drain (active if quick_fix_count >= 2 and fundamental_count == 0), Complexity Creep (active if total side_effects count >= 2 or layers with times_applied > 3), Normalization (active if any layer duration_active contains "month" or "quarter" or "year")
       - `_estimate_automation_coverage(layers)`: estimate = round((len([l for l in layers if l.layer_type != "quick_fix"]) / len(layers)) * 100) -- percentage of non-symptomatic coverage
       - `_generate_recommendations(burden_result, layers)`: 2-4 automation-specific recommendations based on debt_score tiers and active erosion channels. E.g., "Convert manual review to training pipeline" for Knowledge Drain.
    6. Merge burden_result with automation additions, format per response_format

    Use @mcp.tool decorator with `annotations={"readOnlyHint": True, "idempotentHint": True}`.
