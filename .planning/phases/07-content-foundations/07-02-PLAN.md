## Agent Visual Vocabulary

### Design Principles
- Shape-coded nodes: distinct SVG shape per component type (shapes do the heavy lifting for identification)
- Subtle color tints per shape: muted palette colors as secondary reinforcement
- Consistent with dark cybernetic aesthetic (existing palette)
- Same edge/link styles as existing CLD artifacts (curved bezier paths, +/- polarity labels)
- All SVG specs are for inline React rendering (no external CSS)

### Node Type Specifications

For each of the 5 node types, specify:
- Shape name and visual description
- SVG path data (or SVG element type like `<polygon>`, `<rect>`, `<ellipse>`)
- Default dimensions (width x height in px, sized for canvas readability)
- Fill color (muted tint from palette family)
- Stroke color and width
- Label text positioning (centered, font: DM Sans 12px, color: #e8ecf4)
- Icon suggestion (optional simple glyph inside shape)

**1. Agent - Hexagon**
- Shape: Regular hexagon (6 sides)
- SVG: `<polygon>` with 6-point coordinate calculation
- Default size: 80px wide x 70px tall
- Fill: #1a2744 (deep blue-navy, muted from #6b8aff family)
- Stroke: #6b8aff (blue accent), 2px
- Rationale: Hexagon = hub/orchestrator, visually distinct and central

**2. Tool - Rectangle (rounded corners)**
- Shape: Rectangle with border-radius 6px
- SVG: `<rect>` with rx="6" ry="6"
- Default size: 90px wide x 50px tall
- Fill: #1a3a2e (deep teal-green, muted from #00d4aa family)
- Stroke: #00d4aa (teal accent), 2px
- Rationale: Rectangle = functional unit, familiar "block" metaphor

**3. Memory - Cylinder**
- Shape: Cylinder (rectangle with elliptical top/bottom)
- SVG: Composite `<path>` drawing cylinder shape (rect body + ellipse cap)
- Default size: 60px wide x 70px tall
- Fill: #2a1a3a (deep purple, muted from #9b6bff family)
- Stroke: #9b6bff (purple accent), 2px
- Rationale: Cylinder = database/storage, universal data persistence symbol

**4. Evaluator - Diamond**
- Shape: Diamond (rotated square)
- SVG: `<polygon>` with 4 diamond points
- Default size: 70px wide x 70px tall
- Fill: #3a2a1a (deep amber, muted from #ffb86b family)
- Stroke: #ffb86b (amber accent), 2px
- Rationale: Diamond = decision point, familiar from flowcharts

**5. Constraint - Octagon**
- Shape: Regular octagon (8 sides)
- SVG: `<polygon>` with 8-point coordinate calculation
- Default size: 70px wide x 70px tall
- Fill: #3a1a1a (deep red, muted from #ff6b6b family)
- Stroke: #ff6b6b (coral accent), 2px
- Rationale: Octagon = stop sign, universal constraint/limit signal

### Edge Specifications (unchanged from existing)
- Positive polarity (+): Teal (#00d4aa), curved bezier, "+" label at midpoint
- Negative polarity (-): Coral (#ff6b6b), curved bezier, "-" label at midpoint
- Stroke width: 2px
- Arrow: filled triangle at target end, same color as edge
- Label font: JetBrains Mono 11px

### Canvas Background
- Same as existing: #0f1117 (dark charcoal)
- Grid dots: #1e2130 (subtle grid, optional)
